### 2. 导航菜单下划线滑动动画 ✅
**问题**：下划线直接跳转，没有过渡效果

**解决方案**：
```css
.modern-nav .nav-item.active::after {
  transition: all 0.4s cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

**效果**：
- 下划线现在会平滑滑动到新位置
- 使用弹性缓动函数，有轻微回弹效果
- 过渡时间0.4秒，感觉流畅自然

### 3. 附近家政页面卡片动画 ✅
**问题**：卡片全部同时出现，没有动画

**解决方案**：
修改选择器从`.m-card`改为`.card-col`，因为卡片在`el-col`容器中：

```css
.card-col {
  animation: cardFadeIn 0.6s cubic-bezier(0.34, 1.56, 0.64, 1) backwards;
}

.card-col:nth-child(1) { animation-delay: 0.1s; }
.card-col:nth-child(2) { animation-delay: 0.15s; }
.card-col:nth-child(3) { animation-delay: 0.2s; }
/* ... 更多卡片 */
```

**效果**：
- 卡片依次从下方淡入
- 每个卡片延迟0.05秒
- 有弹性回弹效果
- 与月嫂页面动画一致

## 技术细节

### 动画性能优化
```css
/* 使用transform而非position */
transform: translateY(20px);  /* ✅ 好 */
top: 20px;                    /* ❌ 差 */

/* 使用opacity而非visibility */
opacity: 0;                   /* ✅ 好 */
visibility: hidden;           /* ❌ 差 */
```

### 缓动函数
```css
cubic-bezier(0.34, 1.56, 0.64, 1)
```
- 0.34: 开始时加速
- 1.56: 超过目标（产生回弹）
- 0.64: 减速
- 1.0: 回到目标位置

### 动画延迟策略
```css
/* 每个卡片延迟0.05秒 */
:nth-child(1) { animation-delay: 0.1s; }
:nth-child(2) { animation-delay: 0.15s; }
:nth-child(3) { animation-delay: 0.2s; }
```
