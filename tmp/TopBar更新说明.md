## 技术细节

### 定位功能
```javascript
getCurrentLocation() {
  if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(
      (position) => {
        // 获取坐标后可以调用地理编码API转换为城市名
        this.currentCity = '北京'; // 默认城市
        localStorage.setItem('currentCity', this.currentCity);
      },
      (error) => {
        // 定位失败使用默认城市
        this.currentCity = '北京';
      }
    );
  }
}
```
